The W.M.N. Machine

This game is set in a future of disappearing nature. Tune your Wonderful Mysterious Nature Machine to the correct colour frequency, then to the correct natural item. Maybe the future won't be so bleak?

This game is playable on windows or other platforms with Love2D installed.
To build game on windows, run 'build.ps1' using powershell from the 'src' directory. 
To start, run ggj.exe
